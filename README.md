# Tripleten web_project_around_react

This is an interactive React Single-Page Application (SPA) that allows users to update their profile information (username and job title), as well as add, delete, and like photos.

The project uses React Hooks for state and lifecycle management and includes asynchronous API requests to communicate with the backend.

Key features include:

* Editable user profile (name and job title)
* Add, delete, and like photos
* Reusable React components
* API integration with asynchronous requests
